These are the steps to run the program :

        step 1 - Download the zip file from the Repo 
        step 2 - Open the code in the compiler of your choice (Google collab Recommended)
        step 3 - Make sure to chnage the Run time type too T4 Gpu 
        step 4 - Run all the cells individually   
        step 5 - Upload two images (The content image & The style image) 
        step 6 - Use the example_train_config.yaml file for configuration (you can make tweaks to the um_epochs,
learning_rate according to your needs)
      You can make changes to the output size as well
